# Tarefa 16 - React

## Descrição
Projeto de aprendizado em React para o curso EBAC.

## Pré-requisitos
- Node.js instalado
- npm ou yarn

## Instalação
```bash
npm install
```

## Execução
```bash
npm start
```

## Estrutura do Projeto
- `/src` - Código fonte da aplicação
- `/public` - Arquivos públicos

## Tecnologias
- React
- JavaScript
